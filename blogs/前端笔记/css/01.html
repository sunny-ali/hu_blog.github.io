css中几种水平垂直居中的方式

银河系小仙女 2020-01-09 css

在前端开发中经常会让一个盒子模型水平，垂直居中。接下来主要是实现的几种方法

# 1，将body中的第一个div元素水平垂直居中

代码如下：

<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>index</title>  
    <style>  
       html,body{
    		background-color: #f0f0f0;
    	}
        .content {  
            width: 100px;  
            height: 100px;  
            background: pink;  
        }  
    </style>  
</head>  
<body>  
    <div class="content"></div>  
</body>  
</html>

.content {  
            margin: 0 auto;

接下来就是让这个粉红色的div垂直居中， 1，首先我们需要设置div元素的祖先元素html和body的高度为100%(因为它们默认是0)，并清除默认样式，设置margin和padding设置为0（如果不清除默认样式的话，浏览器会出现滚动条） 2，让div往下，往右偏离。默认情况下，元素在文档流里是从上往下、从左往右紧密布局的，我们不可以直接通过top属性改变它的垂直偏移。这就需要使用position属性，设置它的值为relative，就可以通过top,bottm,right,left这些属性使它在正常的文档流中发生位置偏移（注意它并没有脱离文档流，原来的位置还会占据） 设置top:50%,那么div的位置就在整个页面高度的1/2处，明显是偏下了，这时我们需要向上偏移div自身高度的一半才是真正的垂直居中了，设置margin-top:-50px;也可以设置transform: translateY(-50%)，意思是使得div向上平移（translate）自身高度的一半。效果是一样的 代码：

html,body{
           background-color: #f0f0f0;
    		width: 100%;
    		height:100%;
    		margin:0;
    		padding:0;
    	}
.content {  
            width: 100px;  
            height: 100px;  
            background: pink; 
            margin: 0 auto;//水平居中
            //垂直居中
            position: relative;
            top:50%;
            margin-top:-50px;  // 也可以使用 transform: translateY(-50%)
        }

# 2，使div中的子元素(块级元素)水平垂直居中

代码如下：

<head>  
    <meta charset="UTF-8">  
    <title>index</title>  
    <style>  
        .father {  
            width: 300px;  
            height: 300px; 
            background-color: #f0f0f0; 
        } 
        .child{
        	width: 200px;  
            height: 200px; 
            background-color: pink; 
        } 
    </style>  
</head>  
<body>  
   <div class="father">
    	<div class="child">
    		<span>行内元素</span>
    	</div>
    </div>  
</body>  
</html>

        } 
 .child{
            position: absolute;
            top:50%;
            left: 50%;

# 3，使div中的子元素(行内元素)水平垂直居中

代码如上所示，使span标签里面的内容水平垂直居中，则设置span标签的父元素，使行高与高一样

.child{
           height：200px;
            line-height: 200px;
            text-align: center;
        }
